# 用户旅途

> 描述用户从零开始到每天使用的完整体验路径。
> 格式：看到什么 → 做什么 → 看到什么 → 得到什么

---

## 第一次使用

### 打开应用

**看到什么**
配置向导弹出。

**做什么**
按向导一步步操作：

1. 从本机微信群列表中勾选目标群（可多选）
2. 选择策略标签（技术问答群 / 资讯分享群 / 通用群）
3. 可选：用自然语言补充策略，比如「过滤掉闲聊和纯表情，重点关注有明确答案的问题」
4. 确认

**看到什么**
自动触发第一次采集，显示简单加载状态。

**得到什么**
采集完成后进入主界面，看到第一批结果。

---

## 主界面

用户看到一个列表，每条内容是一张卡片。

### 卡片类型

AI 提取内容时自动判断类型，选择对应卡片格式：

| 类型 | 显示字段 |
|------|---------|
| QA 卡片 | 问题标题 + 答案摘要 + 分数 + 简短理由 + 时间 + 来源群 |
| 链接卡片 | 标题 + 链接 + 观点摘要 + 分数 + 简短理由 + 时间 + 来源群 |
| 话题卡片 | 标题 + 内容摘要 + 分数 + 简短理由 + 时间 + 来源群 |

点开单条卡片，展开看完整内容和点评入口。

### 界面布局

- 右上角：采集按钮
- 列表顶部：排序（按时间 / 按分数）+ 筛选（按群 / 按类型）
- 列表：卡片列表，默认按时间倒序
- 卡片上：点评按钮（针对单条）
- 底部：全局对话框（针对整体点评）

---

## 每天使用

### 触发采集

**看到什么**
主界面，右上角有采集按钮。

**做什么**
点采集按钮，或在底部对话框输入「采集今天的消息」。

**看到什么**
简单加载状态。系统自动记录本次采集时刻，下次只拉新消息。

**得到什么**
新内容追加到列表顶部。

---

### 查看结果

**看到什么**
每条卡片带 AI 评分和简短理由，比如：

```
怎么在 Claude 里调用自定义工具？
87分 · 答案完整，有可操作步骤
2026-05-09 14:23  AI大航海
```

**做什么**
浏览列表，按需筛选或排序。

---

### 点评单条

**做什么**
点卡片上的「点评」按钮，输入：「这条答案不完整，没有说清楚怎么配置」

**看到什么**
AI 根据点评完善策略，并提示：「策略已更新，是否立刻重跑？」

**做什么**
选择「立刻重跑」或「下次生效」。

**得到什么**
策略更新，点评沉淀为该群的历史反馈，下次采集自动带入。

---

### 整体点评

**做什么**
在底部全局对话框输入：「今天噪音太多，很多闲聊被提取进来了」

**看到什么**
AI 更新策略，提示是否立刻重跑。

**得到什么**
同上，策略完善 + 历史反馈沉淀。

---
